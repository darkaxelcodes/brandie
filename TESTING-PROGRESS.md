## Testing Approach

### Manual Testing Focus
1. **UI/Visual Testing**: Check all layouts, responsive design, styling
2. **Functional Testing**: Verify all buttons, forms, actions work
3. **Integration Testing**: Test API calls, database operations
4. **User Flow Testing**: Complete end-to-end user journeys
5. **Edge Case Testing**: Test error handling, validation, limits

### Tools & Methods
- Browser: Chrome DevTools for inspection
- Network: Monitor API calls and responses
- Console: Check for errors and warnings
- Database: Verify data persistence via Supabase dashboard
- Performance: Check loading times and responsiveness

### Test Data Requirements
- Test user account with authentication
- Sample brand with partial completion

- Multiple brands for testing selection/switching
- Token balance for testing AI features
- Different screen sizes for responsive testing
